About Anonymize mode

What is Anonymize mode?

Instead of blocking text with placeholders, Anonymize replaces sensitive information with realistic fake equivalents — so your prompts stay readable while your real data stays private.

Veil anonymizes: Names · Emails · Phone numbers · Addresses · Locations · Organizations · Dates of birth · SSNs · Credit card numbers

e.g. Priya Sharma → Claire Bennett    priya@acme.com → claire.b@mailhub.io

If a type can't be anonymized (e.g. API unreachable for that request), Veil automatically falls back to masking it — so nothing ever leaks.
